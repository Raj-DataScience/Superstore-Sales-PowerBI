# Superstore Sales — Power BI Dashboard

## Overview
An interactive Power BI dashboard analyzing $2.26M in retail sales 
across 4 years (2015–2018), 3 product categories, 4 US regions, 
and 3 customer segments.

## Key Insights
- Technology is the top revenue category at $827K
- West region leads all geographies at $725K
- Consumer segment drives 51% of total revenue
- Q4 consistently peaks across all years

## Tools Used
- Microsoft Power BI Desktop
- Power Query for data transformation
- DAX for calculated measures
- Dataset: Superstore Sales (9,800 rows)

## Files
- Superstore_Sales_Power_BI_Report.pbix — Power BI dashboard file
- Superstore_Sales_Report.docx — Full analysis report
